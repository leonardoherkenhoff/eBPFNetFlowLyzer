# Security Policy

## Supported Versions

We only provide security updates for the current major release of **Lynceus**.

| Version | Supported          |
| ------- | ------------------ |
| v2.0.x  | :white_check_mark: |
| < v2.0  | :x:                |

## Reporting a Vulnerability

The **Lynceus** team takes the security of our high-performance engine seriously. If you believe you have found a security vulnerability, please do not open a public issue. Instead, follow these steps:

1.  **Initial Report**: Email the maintainer at `leonardoherkenhoff@gmail.com` with a detailed description of the vulnerability.
2.  **Reproduction**: Include steps to reproduce the issue (e.g., a specific PCAP file or BPF loading command).
3.  **Response**: You will receive an acknowledgment within 48 hours.
4.  **Disclosure**: We will coordinate a fix and a public disclosure date.
